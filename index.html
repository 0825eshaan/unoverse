Setup & Loop Structure

Every Arduino program has these two functions:

void setup() { // Runs once at startup } void loop() { // Runs repeatedly forever }

Digital Input/Output

pinMode(13, OUTPUT); // Set pin as output digitalWrite(13, HIGH); // Turn on (5V) digitalWrite(13, LOW); // Turn off (0V) int state = digitalRead(2); // Read pin state

Analog Input/Output

int value = analogRead(A0); // Read 0-1023 analogWrite(9, 128); // PWM output 0-255

Serial Communication

Serial.begin(9600); // Start serial Serial.println("Hello"); // Print with newline Serial.print(value); // Print without newline
